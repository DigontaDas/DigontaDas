## 🔬 Research & Thesis

### 🫀 Efficient 3D Tiled CNN Architecture for Coronary Artery Stenosis Detection
> **Undergraduate Thesis | BRAC University**

- 🏥 Built an **end-to-end 3D Deep Learning pipeline** for coronary artery segmentation using **ImageCAS dataset** + custom data from **Ibrahim Cardiac Hospital & Bangladesh Medical Hospital**
- 🧩 Implemented a **3D Segmentation Model** with volumetric tiling to process high-resolution CT Angiography volumes
- ⚙️ Engineered a robust preprocessing engine using **Hounsfield Unit (HU) clipping & normalization** to mitigate metal artifacts and air noise
- 📉 Optimized training stability by replacing standard **Dice Loss** with a **Hybrid Combo Loss (Dice + BCE)**, solving the "empty mask" prediction issue in early epochs
- 🩺 Proposing a **clinical decision support system** to quantify calcification (hard plaque) for automated cardiac risk assessment
